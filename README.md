# Calculadora-AngularJS

Este proyecto es una calculadora web simple desarrollada utilizando **AngularJS**. La he creado como un punto de partida para familiarizarme con los conceptos básicos de AngularJS antes de avanzar hacia el estudio y desarrollo con Angular.

## Características

- Interfaz de usuario intuitiva con botones para números, operadores y funciones de cálculo.
- Funcionalidad para realizar cálculos matemáticos básicos, incluyendo sumas, restas, multiplicaciones y divisiones.
- Capacidad para ingresar expresiones matemáticas y obtener resultados instantáneos.
- Función de borrado para eliminar los valores ingresados y comenzar de nuevo.

## Motivación

Experiencia en el manejo de directivas, controladores y en la interacción entre la vista y el modelo de datos en AngularJS.
